Blazing Angels 2: Secret Missions of WWII on Windows 10

1. Download 1.01 patch 
2. Download dinput8.dll
3. Change framerate in registry to 60 decimal or 3c hexadecimal (run regedit as administrator !)
4. Set bin\ba2.exe file windows 7 or XP compatibility and run as administrator
5. Run directly ba2.exe

	happy shooting

PS:Change plane with 'x' in Flying tigers (tactical map key?)
